## Deskripsi app:
MyTaskList App adalah aplikasi manajemen tugas harian yang dirancang dengan antarmuka modern bertema Soft Pink untuk membantu pengguna mengelola produktivitas secara personal. Aplikasi ini mengintegrasikan komponen dasar React Native dengan manajemen state yang kompleks untuk menangani pembuatan, penyuntingan, dan pemantauan progres tugas secara real-time.

## Cara menjalankan project 
1. Pastikan Node.js dan Expo CLI sudah terinstal di perangkat.
2. Buka terminal pada direktori project.
3. Jalankan perintah: npx expo start.
4. Scan QR Code yang muncul menggunakan aplikasi Expo Go di HP fisik (Android/iOS).
5. Pastikan HP dan Laptop berada di jaringan Wi-Fi yang sama agar sinkronisasi berjalan lancar.
